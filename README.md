# Unbeatable Console Pong Game

A simple console Pong game written in C# with an unbeatable opponent.

## Features

- Console-based Pong game
- Unbeatable opponent
- Smooth keyboard controls (W / S)
- Ball collision detection
- Dynamic game speed that increases over time
- Win/Lose screen

## Controls

| Key | Action |
|------|--------|
| W | Move paddle up |
| S | Move paddle down |

## How to Run

1. Clone the repository.
2. Open `Ping-Pong.sln` in Visual Studio.
3. Build and run the project.

## Technologies

- C#
- .NET
- Console Application
